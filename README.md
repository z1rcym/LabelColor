# LabelColor

--インストール方法--
.jsxbin ファイルを ScriptUI Panels フォルダに入れ、ウィンドウから実行します。


--機能--
選択しているレイヤーと同じラベルカラーを持つレイヤーを一括で非表示・表示にできます。
